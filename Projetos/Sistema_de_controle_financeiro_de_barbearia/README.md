Sistema de Controle Financeiro de Barbearia
Projeto desenvolvido em Python com foco em Programação Orientada a Objetos, criado para resolver um problema real do meu dia a dia como barbeiro:
**não ter um controle claro de quanto dinheiro entrava por mês na barbearia.**

Funcionalidades
- Registro de entradas financeiras
- Cálculo do faturamento total
- Organização dos dados
- Execução via terminal

## Problema que o projeto resolve
Antes desse sistema, o controle financeiro era feito de forma manual, o que dificultava:
- saber o faturamento mensal
- acompanhar entradas de dinheiro
- ter uma visão clara do financeiro da barbearia

Este sistema surgiu como forma de **unir estudos em programação com uma necessidade real**.
