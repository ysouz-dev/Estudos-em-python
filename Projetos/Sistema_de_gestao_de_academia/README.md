# Sistema de Gerenciamento de Academia

Projeto desenvolvido em Python com foco em Programação Orientada a Objetos,
simulando o gerenciamento de alunos, funcionários e equipamentos.

## Funcionalidades
- Cadastro de alunos e funcionários
- Listagem e busca por CPF
- Controle de planos
- Gerenciamento de equipamentos

## Objetivo
Projeto de estudo para praticar conceitos de POO como:
- Classes
- Objetos
- Herança
- Encapsulamento
- Polimorfismo
